# aty_sneaker_sales_analytics
1. Краткое описание проекта

Цель: Аналитика продаж онлайн-магазина кроссовок с китайского маркетплейса Poizon. Миграция Excel → SQLite БД → SQL-запросы для revenue, маржи, RFM.

Задача: Из плоского Excel файла → нормализованная БД → бизнес-метрики (топ-бренды, сегментация клиентов, контроль партий).

Результат: ~600k ₽ revenue, 50 заказов, 30 клиентов, 31 товар. Готовый дашборд-код.

2. Стек и требования
БД: SQLite 3 (aty_database.db)

Инструмент: DB Browser for SQLite

Данные: Реальный Excel от магазина
